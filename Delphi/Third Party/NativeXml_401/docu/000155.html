TNativeXml
Hierarchy, Members, Properties, Methods, Events, Unit: NativeXml
TNativeXml = class(TDebugComponent)
Description
TNativeXml is a very fast XML reader (on typical hardware storage 15 Mb per second), because it loads external data in chunks and buffers it in memory. Use Create to create a new instance, use LoadFromFile/LoadFromStream to load the XML document from a file or stream, and use SaveToFile and SaveToStream to save the XML document.
