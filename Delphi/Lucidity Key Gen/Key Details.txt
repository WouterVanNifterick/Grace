The format for a Lucidity Key file is:

======================================
License Key File for Lucidity 1.0
BEGINKEY>>>>
UserName
UserEmail
KeySection01
KeySection02
[...]
KeySection24
KeySection25
Checksum
<<<<ENDKEY
======================================




Each key section is:

	md5(Salt + UserName + Salt + UserEmail + Salt)

The salt used matches the key section. IE. KeySection04 will use the 4th salt. 
  


The Checksum value is calculated by joining all UserName, UserEmail and all key sections. This blob of data is then MD5 hashed. The hash is the "checksum". 


Any data outside the BEGINKEY and ENDKEY tags will be discarded.
